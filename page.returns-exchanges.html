How to return or exchange

Go to the returns portal

Click here to visit our returns & exchanges portal page. Enter your order number and either your shipping code or email address.

Choose what’s next

Pick whether it’s a return or exchange and answer a few questions. You’ll be asked to provide a photo of the item(s). 

Get an RMA Number

If approved, you will be provided the return instructions and a Return Merchandise Authorization (RMA) number from our team via email. This is required for all successful returns.

Pack and ship

Securely pack item(s) in a box and ship them! We encourage you to use a trackable shipping service. Please allow up to 7-14 business days for your return to be processed.
